# 🛒 ShopEase - E-Commerce Flutter App
## Apex Planet Internship - Task 2

### 📌 Project Overview
ShopEase is a Flutter-based E-Commerce application developed as part of the Apex Planet Internship program. Task 2 focuses on implementing the core user interface and integrating Firebase services to create the foundation of a modern e-commerce application.
## 🚀 Task 2 Objectives
- Integrate Firebase with Flutter.
- Create user authentication screens.
- Design product listing screens.
- Implement navigation between screens.
- Set up basic database structure using Cloud Firestore.
- Organize the project using a clean folder structure.
